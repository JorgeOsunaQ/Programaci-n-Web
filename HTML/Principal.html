Descripción del Trabajo

La creación de este conjunto de páginas se realizó con la finalidad de comprender los aspectos básicos de HTML5 y CSS3.
La página que podemos apreciar es una combinación de un menu diseñado para hacer referencia a un iframe que apunta a su vez a distinas páginas. Cada uno de estos archivos html hace referencia a un archivo CSS único y creado para darle diseño a cada página de forma independiente.
